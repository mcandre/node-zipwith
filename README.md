# zipwith - A Node.js port of Haskell's zipWith function

# EXAMPLE

    $ node example.js
    [ 4, 4, 4 ]

# HOMEPAGE

https://github.com/mcandre/node-zipwith

# REQUIREMENTS

* [Node.js](http://nodejs.org/) 0.10+

# INSTALL

    $ npm install zipwith

# LINT

Keep the code tidy:

    $ npm run-script lint
